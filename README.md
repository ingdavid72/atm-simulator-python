# ATM Simulator Python 🏦

Simulador de cajero automático desarrollado en Python. 
Este proyecto permite realizar operaciones bancarias básicas mediante un menú interactivo en consola.

## Funcionalidades

- Consultar saldo
- Depositar dinero
- Retirar dinero
- Validación de fondos disponibles
- Menú interactivo
- Salir del sistema

## Tecnologías utilizadas

- Python 3
