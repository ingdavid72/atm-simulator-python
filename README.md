# ATM Simulator Python 🏦

Simulador de cajero automático desarrollado en Python. 
Este proyecto permite realizar operaciones bancarias básicas mediante un menú interactivo en consola.

## Funcionalidades

- Consultar saldo
- Depositar dinero
- Retirar dinero
- Validación de fondos disponibles
- Menú interactivo
- Salir del sistema

## Tecnologías utilizadas

- Python 3

## Cómo ejecutar el proyecto

1. Clonar el repositorio:
2 Entrar a la carpeta del proyecto:
cd atm-simulator-python
3 Ejecutar el programa:
python main.py
